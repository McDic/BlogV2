이 저장소는 Mkdocs 를 통해 웹사이트로 만들어졌으며, netlify 를 이용하여 배포되었습니다.

## How to run locally

```bash
pip install -r requirements.txt
mkdocs -v serve
```

* `mkdocs serve`: Start the live-reloading docs server.
* `mkdocs build`: Build the documentation site.
